# Project-Admin-Dashboard

A dashboard layout project built as part of The Odin Project's Intermediate HTML and CSS course.

## About

This project focuses on practicing CSS Grid by recreating an admin dashboard layout. The page includes a sidebar, header, project cards, announcements section and trending section.

## Features

- Full dashboard layout using CSS Grid
- Sidebar navigation
- Header with search user info and action buttons
- Project cards section
- Announcements section
- Trending users section
- Google Fonts and Material Symbols
- Basic responsive layout for smaller screens

## Built With

- HTML
- CSS
- CSS Grid
- Google Fonts
- Material Symbols

## What I learned

While building this project, I practised:

- Creating a page layout with CSS Grid
- Nesting grids inside larger grid containers
- Using fixed and flexible grid tacks like 'px', 'fr' and 'minmax()'
- Structuring HTML into clear sections
- Styling cards, navigation, buttons and content panels
- Using external fonts and icons
